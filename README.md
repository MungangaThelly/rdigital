## Target Audience

- 65+ seniors in Sweden
- Family members helping parents
- Libraries and community centers
- Municipalities and care organizations

## Core Learning Tracks

1. **Staying in Touch** - Messaging, email, video calls
2. **Living in Sweden Digitally** - BankID, Swish, e-services
3. **Staying Safe** - Scams, security basics
4. **Confidence Basics** - Understanding your device
5. **Social Media** - Facebook, Instagram basics
